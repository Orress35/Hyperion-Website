Variables and types

A variable is a piece of data stored in the process memory. To declare a variable you have to specify the variable type, name and value: 

    int i = 8            // integer (whole number) variable with name i and value 8
    float f = 1.2     // floating point (decimal) variable with name f and value 1.2

Currently, there's 10 primitive types in Hyperion:

Integers (3):

short - 16-bit integer (-32767 to 32767)
int - 32-bit integer (-2147483647 to 2147483647)
long - 64-bit integer (-9223372036854775807 to 9223372036854775807)

Float (2):

float - 32-bit floating point number (-3.4e+38 to 3.4e+38)
double - 64-bit floating point number (-1.8e+308 to 1.8e+308)

Binary (2):

bit - 0 / 1
byte - 8-bit number (-127 to 127)

Text (2):

char - a single ASCII character
string - a sequence of characters

Logic (1):

bool - true / false
