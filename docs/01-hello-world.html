Hello World

A basic hello world program in Hyperion looks like this: 

    package label
    
    sub start (
        println("hello world")
        exit(0)
    )

The first statement in the file is its package. A package is the path from the project directory to the directory that the file is in.
Next, we have the start subroutine, this is the entry point of the program. Everything inside of it will be executed when the program starts.
Inside of the start subroutine we have two statements: println and exit function calls.
"hello world" is the string that the println function should print
and 0 is the exit code that the exit function should return.
These are function parameters.
