Basic Types:

Integer:

short - 16-bit integer number
int - 32-bit integer number
long - 64-bit integer number


Decimal:

float - floating point number
double - double percision float


Logic:

boolean - true or false
another one coming soon...


Text:

char - ASCII character
string - sequence of characters


Automatic:

func - lets the compiler choose the return type of a function
auto - lets the compiler choose the variable type
var - lets the compiler choose the variable type, can override already declared vars, always declares globally
let - lets the compiler choose the variable type, cannot override other vars, always declares locally





Type Modifiers:

final - defines the variable as read-only
unsigned - defines the variable as positive-only





Operators:

+ - addition
- - subtraction
* - multiplication
/ - division
% - modulo

= - set
:= - declare and set (if already declared, just set)
+= - add and set
-= - subtract and set
*= - multiply and set
/= - divide and set
%= - modulo and set

== - equal to
!= - not equal to
> - greater than
< - less than
>= - greater than or equal to
<= - less than or equal to

&& - logical and
|| - logical or
! - logical not

& - bitwise AND
| - bitwise OR
^ - bitwise XOR
~ - bitwise NOT
<< - bitwise left shift
>> - bitwise right shift

sizeof - returns size of data type
?: - returns value based on condition
& - returns address of variable
. - access data inside class or struct
-> - access data inside pointer





Basic I/O:

print - print text to console
println - print text to console and go to the next line
getInput - get console input





Comments:

// text - single line comment
/* text */ - multi-line comment
