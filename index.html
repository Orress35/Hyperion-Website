What is Hypern?

Hypern is a fast and flexible programming language created in September 2020. Most of its syntax comes from C++, Go, JavaScript and Rust, however, there are lots of unique features which are not present in any of the major programming languages.



Why Hypern?

• Speed

Hypern is faster than all major programming languages including C, C++ and Java, which makes it perfect for any performance-heavy task.

• Flexibility

Hypern is suitable for almost any environment:
Desktop Development - Windows, Linux and MacOS,
Mobile Development - Android,
Cyber Security - encryption, hashing, penetration testing,
Game Development - Unity, Godot, Unreal and custom engines,
Operating Systems - UEFI support
and much more.

• Language Level

While Hypern is a higher-level language, you can still
1. Directly access the virtual memory
2. Call assembly instructions or have whole functions written in pure assembly
3. Access the CPU registers
4. Directly call interrupts or syscalls
5. Directly access devices
