Which branch should you choose?


General Use - RSU:

RSU is a low-level procedural programming language with C-like syntax.
It can be used on all major operating systems (Windows, Linux and MacOS), supports all assembly instructions, supports most C and C++ libraries, has easy-to-use file, math and GUI modules and executes at the same speed as pure C.


Machine Learning and Algorithms - RSL:

RSL is a high-level scripting language based on Python.
It supports C-like syntax, variable type declarations and has a few automatic optimizations to make your Python code faster. Sadly, as of right now, RSL has roughly the same performance as pure Python, so I wouldn't recommend using it until the first stable version comes out.


Mathematics and Data Analysis - RSF:

RSF is a low-level functional language.


Mobile Development - RSM:

RSM is a high-level object-oriented language based on Java.
This branch is currently in early development and I'm planning to fully migrate it from Java to C++ on release, which means that the current description for it will be completely invalid in a few months.
